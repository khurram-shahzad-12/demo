1- clone code from GitHub repository
2- open demo folder 
3- open backend folder with code editor like vs code
4- go to terminal and write command there
npm install   press enter

5- after installation node-modules, again go to terminal and write command
npm run dev   press enter

your server will be start on localhost:5000
-----------------------------------------

6- let the above process continue, minimize the window ajd again open demo folder, inside open frontend folder with code editor

7- open it's terminal and write command npm install
8-after installation write command 
npm run dev   press enter

your app will start on localhost:5173

-----------------------------------------
now open browser and open link 
http://localhost:5173

your web app will start working
